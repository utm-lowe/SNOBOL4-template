# SNOBOL4-Template
This is a codespace for developing in SNOBOL4. It comes withe a copy of 
[cnsobol4](https://www.regressive.org/snobol4/csnobol4/curr/) installed in 
.local. The source code is in .local/src directory.

You can run the snobol4 interpreter by the command `snobol4`, though if you don't
want it to display the detailed banner, use `snobol4 -b`.
